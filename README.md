# SFI Object 13 — «Доктор»

Атмосферный сайт-досье SFI об Объекте №‑13 «Доктор» с отдельным
зашифрованным разделом «Поглощение Скверной» и тринадцатью стадиями мутации.

[Открыть сайт на GitHub Pages](https://roman161rusrosrov-cmyk.github.io/sfi-object-13-doctor/)

## Состав

- основное зашифрованное досье;
- отдельный зашифрованный протокол MOR‑13;
- аномальные способности и сбалансированные ограничения;
- 13 самостоятельных изображений стадий мутации;
- адаптивное оформление для компьютеров и телефонов;
- автоматическая статическая сборка и публикация через GitHub Pages.

## Локальный запуск

Требуется Node.js 22 или новее.

```bash
npm ci
npm run dev
```

Проверка производственной сборки:

```bash
npm test
```

## Безопасность

Пароли, токены, приватные ключи, файлы окружения и
`.openai/hosting.json` в репозиторий не включены. Содержимое закрытых разделов
хранится только в виде AES‑256‑GCM-шифротекста и расшифровывается локально в
браузере.

Публичный статический сайт не заменяет серверную систему авторизации:
зашифрованные данные можно скачать и пытаться подбирать пароль офлайн. Для
действительно конфиденциальных материалов нужен серверный контроль доступа.
