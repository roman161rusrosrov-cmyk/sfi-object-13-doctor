# SFI Object 13 — «Доктор»

Интерактивное досье SFI на Объект №13 — Йенса «Доктора».

## Возможности

- две независимые зашифрованные зоны доступа;
- 13 стадий мутации и поглощения Скверной;
- 13 аномальных способностей со сбалансированными ограничениями;
- отдельная иллюстрация для каждой стадии;
- адаптивный интерфейс в стилистике закрытого медицинского архива SFI.

Пароли доступа распространяются отдельно и не хранятся в открытом исходном коде.

## Запуск

Требуется Node.js 22.13 или новее.

```bash
npm ci
npm run dev
```

## Сборка

```bash
npm run build
```
